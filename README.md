# Pace-Clock
Pace Clock code for Arduino Nano 5V
Written in C++

## Parts Required:
- Arduino Nano 5V
- 4-digit 7-segment TM1637
- Battery Back
- Low dropout DC converter
- External Square Rocker Switch (SPST)
